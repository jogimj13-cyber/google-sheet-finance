# Google Sheet Financial Assistant
An agent that can edit, and find Spreadsheets in google Drive and can use yahoo Finance.

# Requirements
**1.** Need python installed
**2.** Need openrouter API key(or whatever suits you)
**3.** Need the following libraries installed
   - langchain_openai
   - langchain.agents
   - langchain.messages
   - langchain.checkpoint.memory
   - langchain_mcp_adapters.client
   - asyncio
**4.** Google Sheets and Drive API key enabled(visit https://console.cloud.google.com/)
**5.** Done!
